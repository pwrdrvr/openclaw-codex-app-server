# @pwrdrvr/openclaw-app-server

Codex App Server as an OpenClaw plugin.

This package is cut against the current OpenClaw plugin seam with:

- native plugin commands
- bound inbound conversation claiming
- public channel bindings
- Telegram and Discord interactive callbacks
- Telegram and Discord typing and conversation controls

For local development against an OpenClaw source tree:

```bash
pnpm openclaw plugins install --link /Users/huntharo/gihub/openclaw-app-server
```
